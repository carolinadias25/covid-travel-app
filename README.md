## roamSafe

An app to help you travel internationally, covid secure. Built using React, Firebase and external APIs, to provide users with all the relevant COVID-19 information for each country before travel.

## Technologies

* React
* JavaScript
* HTML
* CSS
* Sass
* Google Firebase
* APIs

## Packages

* Axios
* PropTypes
